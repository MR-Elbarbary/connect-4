# Connect 4

## About

This project is an assigment for intro to AI

> [!IMPORTANT]
> In this version the game ends when the board is full
but in the real connect 4, the game ends when a player connect 4 chesmen (parts) or the board is full

### Tech stack

- React js
- Vite

## How to start

```
cd connect-4
npm install
npm run dev
```
